The following lines shortly describe the key features of the "Investigate a
dataset" project, part of the Data Analyst Nanodegree program by Udacity
(09.2018-04.2019).

The task was to choose a dataset and pose 4 questions, then perform the data
wrangling and exploratory data analysis.

The project was written in Python 3, through Jupyter Notebook. The template was
provide by Udacity with basic information on what is sought in the project.

Please note that this is one of the first projects and thus the quality of the
code and artwork leaves plenty of room for improvement.

The author tried to keep the artwork simple and informative.
